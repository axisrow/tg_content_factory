# TG Agent

**Telegram Post Search, Monitoring & Content Automation**

TG Agent — это self-hosted платформа для мониторинга Telegram-каналов, поиска по сообщениям, генерации контента и автоматизации публикаций. Работает через CLI, Web UI и встроенный AI-агент с 100+ инструментами.

## Возможности

| Модуль | Описание |
|--------|----------|
| **Каналы** | Добавление, сбор, фильтрация сотен каналов |
| **Поиск** | FTS5 полнотекстовый + семантический поиск через векторные эмбеддинги |
| **Контент-пайплайны** | Генерация контента через LLM → модерация → авто-публикация |
| **AI-агент** | 100+ MCP-инструментов, работает через Claude / DeepAgents |
| **Планировщик** | Периодический сбор, поисковые уведомления, расписание публикаций |
| **Фото-загрузчик** | Батчевая отправка фото, автоматическая очередь |
| **Аналитика** | Тренды, топ-сообщения, статистика каналов, контент-аналитика |
| **My Telegram** | Управление диалогами, сообщениями, участниками прямо из UI |

## Интерфейсы

Каждая операция доступна через **три** интерфейса:

=== "CLI"
    ```bash
    python -m src.main channel list
    python -m src.main search "запрос" --limit 20
    python -m src.main pipeline run 1
    ```

=== "Web UI"
    Запустите `python -m src.main serve` и откройте `http://localhost:8000`

=== "AI Agent"
    ```bash
    python -m src.main agent chat "собери последние сообщения из канала @example"
    ```
